### Support Desk App

Fullstack web application that mocks an online IT support ticket system. Built using
Mongo DB, Express.js, Node.js, and React.js.

# Usage

### Install backend dependencies

```bash
npm install
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Run

```bash
npm run dev
```
